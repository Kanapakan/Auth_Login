## Getting Started Find Food Fit Me Application

### **Prerequisites**

* Android Simulator or Android Devices
* recommended Android Pixel 2 (5.0 inches)

### **How to run**

```
git clone https://github.com/Kanapakan/Find-Food-Fit-Me.git
```

```
yarn install
```

```
yarn start
```
